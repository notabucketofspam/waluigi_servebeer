Recommended download process:
  1. Copy the Public folder to your personal Google Drive; click the arrow next to Public at the top (after "Shared with me") and select "Add to My Drive."
  2. Install the "Backup and Sync from Google" application on your PC; https://www.google.com/drive/download/backup-and-sync/ and make sure to login with whichever Google account you copied the Public folder to.
  3. Sync the Public folder to your PC so that you have a local copy; under "Preferences," in the second tab labeled "Google Drive," select the "Sync My Drive to this computer" checkbox and select Public under the "Sync only these folders" option. The files will be saved in the directory after "Folder location."
  4. Sidenote: this process is necessary to avoid file name corruption, since a zip archive file (the default download format from the Google Drive website interface) is largely unable to handle non-alphanumeric characters in compressed file names. This process ensures that all files are properly downloaded in their native format. [edit] This also preserves file metadata such as song titles, artists, albums, etc
